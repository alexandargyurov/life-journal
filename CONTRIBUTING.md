## Contributing to Life Tracker

👍🎉 First off, thank you for taking the time to contribute! 🎉👍

Suggestions and pull requests are highly encouraged!

#### Getting Started:

```sh
git clone git@github.com:alexandargyurov/lifetracker-app.git
cd lifetracker-app
yarn install
yarn start # Get you expo app ready on your phone or emulator 
```

Any questions, let me know!

Thank you 🙌 🙌
